# 飞书扩展 (extensions/feishu/)

[根目录](../../CLAUDE.md) > [extensions](../CLAUDE.md) > **feishu**

## 模块职责

提供飞书/Lark 企业协作平台的适配器，支持消息发送、富文本卡片、文档编辑、云盘集成和 Wiki 管理。飞书是中国领先的企业协作工具。

## 关键功能

### 消息收发

```typescript
// 发送文本消息
await feishu.send(target, "Hello, Feishu!");

// 发送富文本卡片
await feishu.sendCard(target, {
  type: "card",
  elements: [{ type: "text", content: "Hello from OpenClaw!" }],
});

// 发送私信
await feishu.sendDirectMessage(userId, "Hello privately!");
```

### 文档操作

```typescript
// 创建文档
const doc = await feishu.createDocument({
  title: "OpenClaw Report",
  content: "# Report\nGenerated by AI",
});

// 获取文档内容
const content = await feishu.getDocument(docId);

// 更新文档
await feishu.updateDocument(docId, {
  operations: [{ insert: "New content", position: 0 }],
});
```

### 表格操作

```typescript
// 创建表格
const sheet = await feishu.createSheet({
  title: "Data Sheet",
  rows: 100,
  columns: 10,
});

// 写入数据
await feishu.writeCell(sheetId, "A1", "Value");

// 批量写入
await feishu.batchWrite(sheetId, "A1:C3", [
  ["A1", "B1", "C1"],
  ["A2", "B2", "C2"],
]);
```

### 云盘集成

```typescript
// 上传文件
await feishu.uploadFile({
  fileName: "document.pdf",
  fileContent: fileBuffer,
  folderId: rootFolderId,
});

// 下载文件
const file = await feishu.downloadFile(fileId);
```

## 常见问题 (FAQ)

### Q: 如何创建飞书应用？

A: 访问 https://open.feishu.cn/app 创建应用，配置权限后发布。

### Q: 消息发送失败 401？

A: 检查 appId、appSecret 是否正确，确认应用已启用。

### Q: 如何处理region差异？

A: 中国使用 `cn`，国际使用 `us` 或 `eu`。

### Q: 文档协作支持实时吗？

A: 飞书 SDK 支持获取文档，但实时协作需要使用文档链接。

### Q: 如何配置 Webhook？

A: 在飞书开放平台配置事件订阅 URL，接收消息回调。
