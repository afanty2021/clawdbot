# Shared Test Helpers (shared/)

[根目录](../../CLAUDE.md) > [extensions](../) > **shared**

## 模块职责

共享测试工具扩展，提供扩展插件测试所需的通用测试辅助函数。

## 功能说明

### resolve-target-test-helpers

提供 `installCommonResolveTargetErrorCases` 函数，用于测试目标解析（resolve-target）功能的常见错误场景：

- **规范化失败测试**: 测试 allowlist 在隐式模式下的规范化失败情况
- **无目标测试**: 测试未提供目标时的错误处理
- **空 allowlist 测试**: 测试无 allowlist 时的错误处理
- **空白目标测试**: 测试仅包含空白字符的目标处理

### 类型定义

```typescript
type ResolveTargetMode = "explicit" | "implicit" | "heartbeat";

type ResolveTargetResult = {
  ok: boolean;
  to?: string;
  error?: unknown;
};

type ResolveTargetFn = (params: {
  to?: string;
  mode: ResolveTargetMode;
  allowFrom: string[];
}) => ResolveTargetResult;
```

### 使用示例

```typescript
import { installCommonResolveTargetErrorCases } from "@openclaw/shared";

describe("resolveTarget", () => {
  installCommonResolveTargetErrorCases({
    resolveTarget: yourResolveTargetFunction,
    implicitAllowFrom: ["telegram", "discord"],
  });
});
```
