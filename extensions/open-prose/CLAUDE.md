# OpenProse 扩展

[返回目录](../CLAUDE.md) > **extensions/open-prose**

## 模块职责

OpenProse VM 技能包插件，提供斜杠命令和遥测功能。

## 功能概述

### 技能包交付
- **插件 shipped**：通过插件系统交付技能
- **斜杠命令**：支持斜杠命令调用
- **遥测支持**：内置遥测功能

### OpenProse VM
- **虚拟机**：OpenProse 虚拟机环境
- **技能执行**：在 VM 中执行技能

## 关键依赖

### OpenClaw Plugin SDK
- `openclaw/plugin-sdk` - 插件开发工具包

## 使用说明

### 技能交付
OpenProse 技能通过插件系统自动交付，无需手动安装。

### 斜杠命令
插件注册后自动注册斜杠命令：
```
/prose <command>
```

## 注意事项

1. **技能交付**：OpenProse 通过插件系统交付，不是独立安装
2. **VM 环境**：技能在 OpenProse VM 中执行
3. **遥测数据**：默认启用遥测功能
