## Linq Blue API 概述

### 什么是 Linq？

Linq（曾用名 Linq Blue）是一项服务，提供可通过 API 发送 iMessage 的能力，无需 Mac 设备或 Apple ID 配置。

### 核心功能

- **iMessage 发送**：向任意 iPhone 用户发送 iMessage
- **群组支持**：支持群组消息
- **媒体支持**：支持图片、视频等媒体附件
- **反应支持**：支持消息反应（如 👍、❤️ 等）
- **接收消息**：支持接收 iMessage 回复

### 优势对比

| 方案 | 硬件需求 | 配置复杂度 | 稳定性 |
|-----|---------|-----------|--------|
| **Linq API** | 无 | 简单 | 高 |
| BlueBubbles | Mac Mini | 中等 | 中 |
| AppleScript | Mac | 复杂 | 低 |

## API 接口设计

### LinqRuntime 接口
```typescript
interface LinqRuntime {
  // 消息监控
  monitor: ChannelMonitor;

  // 消息发送
  sender: {
    sendMessage(target: LinqTarget, message: OutboundMessage): Promise<LinqMessageResult>;
    sendMedia(target: LinqTarget, media: MediaPayload): Promise<LinqMessageResult>;
    sendReaction(target: LinqTarget, messageId: string, reaction: string): Promise<void>;
  };

  // 目标解析
  targets: {
    parse(target: string): LinqTarget;
    validate(phoneNumber: string): boolean;
  };

  // 账户管理
  accounts: {
    list(): Promise<LinqAccount[]>;
    add(account: LinqAccountConfig): Promise<LinqAccount>;
    remove(accountId: string): Promise<void>;
  };
}
```

### 数据类型

```typescript
interface LinqTarget {
  type: "individual" | "group";
  phoneNumber: string;      // +1234567890 格式
  groupId?: string;         // 群组 ID
}

interface LinqMessage {
  id: string;
  from: string;
  to: string;
  text?: string;
  attachments?: LinqAttachment[];
  reactions?: LinqReaction[];
  timestamp: Date;
}

interface LinqAttachment {
  id: string;
  type: "image" | "video" | "file";
  url: string;
  mimeType: string;
  size: number;
}
```

## 常见问题 (FAQ)

### Q: Linq API 是否需要 Mac？
A: 不需要。Linq 是纯云服务，通过 API 调用发送 iMessage。

### Q: Linq 支持哪些消息类型？
A: 支持文本、图片、视频、文件、位置、联系方式等。

### Q: 如何获取 Linq API 密钥？
A: 访问 https://linq.blue 注册账户并获取 API 密钥。

### Q: Linq 消息是真正的 iMessage 吗？
A: 是的，通过 Linq 发送的消息会显示为蓝色气泡（iMessage），而非绿色气泡（SMS）。

### Q: 支持接收回复吗？
A: 支持。Linq 提供 Webhook 回调，可接收用户回复。

### Q: 费用如何？
A: Linq 按消息计费，具体价格请参考官方定价页面。
