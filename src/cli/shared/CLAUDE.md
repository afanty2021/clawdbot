## 核心功能

### 1. 端口解析 (`parse-port.ts`)

**导出**: `function parsePort(raw: unknown): number`

解析并验证端口号。

**参数**:
- `raw`: 未知类型的输入值（string | number）

**返回**: 端口号（1-65535）

**抛出**:
- `Error`: 如果端口无效或超出范围

**验证规则**:
```typescript
// 1. 类型检查
const port = typeof raw === "string" ? parseInt(raw, 10) : raw;
if (typeof port !== "number" || !Number.isFinite(port)) {
  throw new Error(`invalid port: ${raw}`);
}

// 2. 范围检查
if (port < 1 || port > 65535) {
  throw new Error(`port out of range: ${port}`);
}

return port;
```

**使用示例**:
```typescript
import { parsePort } from "./shared/parse-port.js";

parsePort(18789);           // 18789
parsePort("18789");         // 18789
parsePort(0);               // Error: port out of range: 0
parsePort(65536);           // Error: port out of range: 65536
parsePort("invalid");       // Error: invalid port: invalid
```
