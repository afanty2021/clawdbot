# Browser CLI Actions Input 子模块

> 更新时间：2026-02-20

本模块注册浏览器操作输入命令，包括导航、元素交互、文件处理和表单操作。

## 模块概览

```
src/cli/browser-cli-actions-input/
├── register.ts                  # 命令注册（主入口）
├── register.navigation.ts       # 导航命令
├── register.element.ts          # 元素操作命令
├── register.files-downloads.ts  # 文件和下载命令
├── register.form-wait-eval.ts   # 表单、等待和评估命令
└── shared.ts                    # 共享工具函数
```

## 核心功能

### 1. 命令注册 (`register.ts`)

**导出**: `registerBrowserActionInputCommands(browser, parentOpts)`

注册所有浏览器操作输入命令：
- 导航命令（go、back、forward、reload）
- 元素命令（click、type、select、hover）
- 文件命令（upload、download）
- 表单命令（fill、submit）
- 等待命令（wait、waitForSelector）
- 评估命令（eval、evaluate）

### 2. 导航命令 (`register.navigation.ts`)

**子命令**:
- `browser go <url>` - 导航到 URL
- `browser back` - 后退
- `browser forward` - 前进
- `browser reload` - 刷新页面

**选项**:
| 选项 | 类型 | 说明 |
|------|------|------|
| `--wait` | string | 等待时间（如 "5s"） |
| `--timeout` | string | 超时时间 |

```typescript
// 示例：导航到 URL 并等待加载
openclaw browser go https://example.com --wait 5s
```

### 3. 元素操作 (`register.element.ts`)

**子命令**:
- `browser click <selector>` - 点击元素
- `browser type <selector> <text>` - 输入文本
- `browser select <selector> <value>` - 选择选项
- `browser hover <selector>` - 悬停
- `browser drag <from> <to>` - 拖拽

**选择器类型**:
- CSS 选择器：`#id`, `.class`, `[attr=value]`
- XPath：`xpath=/div[@class='example']`
- 文本：`text=Click me`
- Aria：`aria-label=Submit`

```typescript
// 示例：点击按钮
openclaw browser click '#submit-button'

// 示例：输入文本
openclaw browser type '#username' 'myusername'
```

### 4. 文件和下载 (`register.files-downloads.ts`)

**子命令**:
- `browser upload <selector> <file>` - 上传文件
- `browser download <pattern>` - 下载文件
- `browser downloads` - 列出下载

**选项**:
| 选项 | 类型 | 说明 |
|------|------|------|
| `--pattern` | string | 文件匹配模式 |
| `--save` | string | 保存路径 |

```typescript
// 示例：上传文件
openclaw browser upload '#file-input' '/path/to/file.pdf'

// 示例：下载匹配的文件
openclaw browser download '*.pdf' --save ~/Downloads
```

### 5. 表单和评估 (`register.form-wait-eval.ts`)

**表单子命令**:
- `browser fill <json>` - 填充表单
- `browser submit <selector>` - 提交表单

**等待子命令**:
- `browser wait <duration>` - 等待时间
- `browser waitForSelector <selector>` - 等待选择器
- `browser waitForHidden <selector>` - 等待隐藏

**评估子命令**:
- `browser eval <code>` - 执行 JavaScript
- `browser evaluate <fn>` - 评估函数

```typescript
// 示例：填充表单
openclaw browser fill '{"username": "user", "password": "pass"}'

// 示例：等待元素出现
openclaw browser waitForSelector '.loaded' --timeout 30s

// 示例：执行 JavaScript
openclaw browser eval 'document.title'
```

### 6. 共享工具 (`shared.ts`)

**导出**:
- `parseSelector(raw)` - 解析选择器
- `parseDuration(raw)` - 解析持续时间
- `formatSelector(selector)` - 格式化选择器

## 选择器语法

```typescript
// 支持的选择器格式
interface ParsedSelector {
  type: "css" | "xpath" | "text" | "aria";
  value: string;
  attributes?: Record<string, string>;
}

// 示例
"button.submit"              // CSS
"xpath=//button[@type='submit']"  // XPath
"text=Submit"                 // 文本
"aria-label=Submit form"      // Aria
```
