# Gateway CLI 子模块

> 更新时间：2026-02-20

本模块实现 `openclaw gateway` 命令的所有子命令，用于管理 OpenClaw WebSocket 网关服务。

## 模块概览

```
src/cli/gateway-cli/
├── call.ts              # RPC 调用辅助函数
├── discover.ts          # 服务发现（Bonjour/DNS）
├── register.ts          # 命令注册（主入口）
├── run.ts               # gateway run 子命令
├── run-loop.ts          # 网关运行循环
├── dev.ts               # 开发模式支持
├── shared.ts            # 共享工具函数
└── types.ts             # 类型定义
```

## 核心功能

### 1. 命令注册 (`register.ts`)

**导出**: `registerGatewayCli(program: Command)`

注册所有 Gateway 子命令：
- `gateway run` - 运行网关服务器
- `gateway status` - 显示服务状态
- `gateway discover` - 发现网络中的网关
- `gateway start/stop/restart` - 服务管理

```typescript
export function registerGatewayCli(program: Command) {
  const gateway = program
    .command("gateway")
    .description("Run, inspect, and query the WebSocket Gateway");

  addGatewayRunCommand(gateway);
  addGatewayServiceCommands(gateway); // start/stop/restart
  // ...
}
```

### 2. 服务发现 (`discover.ts`)

**导出**:
- `discoverGatewayBeacons(opts)` - 发现本地和广域网网关
- `parseDiscoverTimeoutMs(raw, fallback)` - 解析超时参数
- `dedupeBeacons(beacons)` - 去重信标
- `renderBeaconLines(beacon, rich)` - 渲染信标信息

**发现方式**:
- **本地**: mDNS/Bonjour (`_openclaw-gateway._tcp`)
- **广域**: DNS TXT 记录 (Tailscale)

```typescript
export function pickBeaconHost(beacon: GatewayBonjourBeacon): string | null {
  // 优先使用解析的服务端点而非 TXT 记录
  const host = beacon.host || beacon.tailnetDns || beacon.lanHost;
  return host?.trim() ? host.trim() : null;
}
```

### 3. 网关运行 (`run.ts`, `run-loop.ts`)

**导出**:
- `addGatewayRunCommand(command)` - 添加 run 子命令
- `runGatewayLoop(opts)` - 网关运行循环

**运行选项**:
| 选项 | 类型 | 说明 |
|------|------|------|
| `--port` | number | 网关端口（默认 18789） |
| `--bind` | string | 绑定地址（默认 127.0.0.1） |
| `--token` | string | RPC 认证令牌 |
| `--auth` | string | 认证模式（token/password/tls/null） |
| `--password` | string | 密码认证 |
| `--tailscale` | string | Tailscale 模式 |
| `--ws-log` | string | WebSocket 日志样式 |
| `--dev` | boolean | 开发模式 |

**运行流程**:
1. 解析选项（继承父命令值）
2. 端口检查和冲突解决
3. 创建网关锁（防止多实例）
4. 启动 WebSocket 服务器
5. 进入事件循环

**端口解析**:
```typescript
export function parsePort(raw: unknown): number {
  const port = typeof raw === "string" ? parseInt(raw, 10) : raw;
  if (typeof port !== "number" || !Number.isFinite(port)) {
    throw new Error(`invalid port: ${raw}`);
  }
  if (port < 1 || port > 65535) {
    throw new Error(`port out of range: ${port}`);
  }
  return port;
}
```
